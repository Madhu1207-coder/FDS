## 🚀 Overview

The **Document Clustering Toolkit** is a three-module project aimed at implementing document clustering techniques using machine learning. It covers custom datasets, built-in datasets like Iris, and integrates with a professional web interface for practical deployment and visualization.

---

## 🌟 Features

- 📄 Cluster real-world textual data
- 🌼 Clustering implementation using the **Iris dataset**
- 🌐 Interactive website to display clustering outputs
- 📊 Visual representation of clusters
- 🧾 Clean, modular structure suitable for educational and project purposes

- 🌸 **Iris Dataset** for Module 2

---

## 🧪 Module Flow

### 📁 Module 1: Custom Dataset Clustering

1. Load and clean a textual dataset.
2. Transform into numerical format (TF-IDF).
3. Apply clustering algorithms.
4. Visualize the results with color-coded clusters.

---

### 🌸 Module 2: Iris Dataset via Google Colab

1. Import built-in Iris dataset.
2. Normalize and analyze features.
3. Apply clustering models (e.g., KMeans).
4. Plot decision boundaries and output clusters.

---

### 🌐 Module 3: Document Clustering Web Page

1. Build a simple website layout using HTML & CSS.
2. Present insights, methods, and clustering visuals.
3. Display tabular outputs and summaries interactively.
